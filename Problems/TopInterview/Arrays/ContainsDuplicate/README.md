# Contains Duplicates

## Question

Given an integer array ```nums```, return ```true``` if any value appears at least twice in the array, and return ```false``` if every element is distinct.

## Contrains

- ```1 <= nums.length <= 3 * 10^5```
- ```-10^9 <= nums[i] <= 10^9```

### Example 1

```markdown

INPUT: nums = [1,2,3,1]

OUTPUT: true
```

### Example 2

```markdown

INPUT: nums = [1,2,3,4]

OUTPUT: false
```

### Example 3

```markdown

INPUT: nums = [1,1,1,3,3,4,3,2,4,2]

OUTPUT: true
```
